# Library-Management-System-Using-Switch-Statement-in-Java
This is a console-based Library Management System developed using Core Java, designed to manage books and students in a library environment. The application uses a menu-driven approach with switch statements to perform operations such as adding books, searching books, registering students, and handling book check-in/check-out.

**Features**
Add, search, and display books
Upgrade book quantity and track availability
Register students and manage borrowing (up to 3 books per student)
Check-out and check-in books
Menu-driven interface using switch-case and loops

**Project Structure**
Library.java – Main class handling menu and user interaction
book.java – Represents individual book details
books.java – Manages book collection and operations
student.java – Represents student details
students.java – Manages student records and borrowing logic

 **Concepts Used**
Core Java
OOP principles (Classes, Objects)
Arrays and ArrayList
Switch-case & loops
Input validation
Menu-driven application design

**How to Run**
Compile all .java files
Run Library.java
Follow on-screen menu options
